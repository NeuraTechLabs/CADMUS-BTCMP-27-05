Provisioning notes and template

This repository includes a minimal `template/topology.yml` adapted from
https://github.com/sderziotis/btcmp-18-mod3. The original repository contains
extensive Ansible provisioning playbooks under `provisioning/` which target
Linux machines and many simulated web services.

I did not copy the full provisioning playbooks to avoid bloating this repo.
If you want the original provisioning playbooks, I can import them or create
an adapted Windows `provisioning/playbook.yml` that runs on `windows-10-x86_64`.
